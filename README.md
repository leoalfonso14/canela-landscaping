# 📖 Owner's Handbook: Launching Your Website (Free Method)

Welcome! To get your website live with **zero monthly hosting fees**, please follow this simple, step-by-step guide. We’ve designed this for non-technical owners to make the handoff as smooth as possible.

---

## 🛠️ Phase 1: Setting up GitHub (Your Code)

GitHub is where your website's "blueprints" (the code) are stored. By creating a personal account, you own the code while allowing your developer to work on it for free.

1. **Create an Account**: Sign up for a free personal account at [github.com](https://github.com).
2. **Create a Private Repository**: Click the **+** (New) button. Name it "CanelaLandscaping". Ensure you select **Private**.
3. **Invite Your Developer**: Go to the **Settings** tab of your new repository, click **Collaborators** on the left, and invite **@leoalfonso14** (or `leandroalfonso14@gmail.com`).

---

## 🛠️ Phase 2: Setting up Supabase (Your Database)

Supabase stores the "Contact Us" submissions from your customers. This ensures you never miss a lead.

1. **Create an Account**: Sign up at [supabase.com](https://supabase.com).
2. **Create a Project**: Click **+ New Project**. Name it "Canela Landscaping". 
3. **Set a Password**: Choose a strong database password and keep it safe!
4. **Invite Your Developer**: Go to **Settings > Team** and invite `leandroalfonso14@gmail.com` as an **Owner**.

---

## 🛠️ Phase 3: Setting up Vercel (Your Free Hosting)

Vercel is the "home" of your website. The **Hobby Plan is 100% Free** and allows you to use your own domain name (e.g., `canelalandscaping.com`) at no extra cost.

1. **Create an Account**: Sign up at [vercel.com](https://vercel.com) using your GitHub account.
2. **Import Project**: Select **Add New Project** and pick the "Canela Landscaping" repository you just created.
3. **Add Your Domain**: Once deployed, go to **Settings > Domains** in Vercel to connect your official business address. Adding a domain on the Hobby plan is **Free**.

---

## 🛠️ Phase 4: Setting up Resend (Your Emails)

Resend allows the website to send you an email notification every time a customer fills out your contact form.

1. **Create an Account**: Sign up at [resend.com](https://resend.com).
2. **API Key**: Go to the **API Keys** tab on the left.
3. **Generate Key**: Click **+ Create API Key**. Name it "Website Notifications".
4. **Send to Developer**: Copy the key (it starts with `re_`) and send it safely to your developer.

# 📖 Manual del Propietario: Lanzamiento de su Sitio Web (Método Gratuito)

¡Bienvenido! Para poner en marcha su sitio web con **cero costos mensuales de alojamiento**, siga esta guía sencilla paso a paso. Hemos diseñado esto para propietarios no técnicos para que la entrega sea lo más fluida posible.

---

## 🛠️ Fase 1: Configuración de GitHub (Su Código)

GitHub es donde se almacenan los "planos" (el código) de su sitio web. Al crear una cuenta personal, usted es el dueño del código mientras permite que su desarrollador trabaje en él de forma gratuita.

1. **Crear una Cuenta**: Regístrese para obtener una cuenta personal gratuita en [github.com](https://github.com).
2. **Crear un Repositorio Privado**: Haga clic en el botón **+** (New). Nómbrelo "CanelaLandscaping". Asegúrese de seleccionar **Private**.
3. **Invitar a su Desarrollador**: Vaya a la pestaña **Settings** de su nuevo repositorio, haga clic en **Collaborators** a la izquierda e invite a **@leoalfonso14** (o `leandroalfonso14@gmail.com`).

---

## 🛠️ Fase 2: Configuración de Supabase (Su Base de Datos)

Supabase almacena los mensajes de "Contacto" de sus clientes. Esto asegura que nunca pierda un cliente potencial.

1. **Crear una Cuenta**: Regístrese en [supabase.com](https://supabase.com).
2. **Crear un Proyecto**: Haga clic en **+ New Project**. Nómbrelo "Canela Landscaping". 
3. **Establecer una Contraseña**: ¡Elija una contraseña de base de datos segura y guárdela bien!
4. **Invitar a su Desarrollador**: Vaya a **Settings > Team** e invite a `leandroalfonso14@gmail.com` como **Owner** (Propietario).

---

## 🛠️ Fase 3: Configuración de Vercel (Su Alojamiento Gratuito)

Vercel es el "hogar" de su sitio web. El **Plan Hobby es 100% gratuito** y le permite usar su propio nombre de dominio (por ejemplo, `canelalandscaping.com`) sin costo adicional.

1. **Crear una Cuenta**: Regístrese en [vercel.com](https://vercel.com) usando su cuenta de GitHub.
2. **Importar Proyecto**: Seleccione **Add New Project** y elija el repositorio "Canela Landscaping" que acaba de crear.
3. **Añadir su Dominio**: Una vez desplegado, vaya a **Settings > Domains** en Vercel para conectar su dirección comercial oficial. Añadir un dominio en el plan Hobby es **Gratis**.

---

## 🛠️ Fase 4: Configuración de Resend (Sus Correos Electrónicos)

Resend permite que el sitio web le envíe una notificación por correo electrónico cada vez que un cliente complete su formulario de contacto.

1. **Crear una Cuenta**: Regístrese en [resend.com](https://resend.com).
2. **Clave API**: Vaya a la pestaña **API Keys si la izquierda.
3. **Generar Clave**: Haga clic en **+ Create API Key**. Nómbrela "Website Notifications".
4. **Enviar al Desarrollador**: Copie la clave (comienza con `re_`) y envíela de forma segura a su desarrollador.

## 🚀 Tech Stack (For Developers)

- **Framework**: React 18 (Vite)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router v6
- **Database**: Supabase
- **Internationalization**: i18next

## 📁 Project Structure

- `/src/components`: Reusable UI components (Navbar, Footer, ServiceCards).
- `/src/pages`: Page-level components (Home, About, Services, Gallery, Contact).
- `/src/supabase`: Centralized database and auth services.
- `/src/locales`: Translation files (English & Spanish).
- `/src/assets`: Static images and global styles.
